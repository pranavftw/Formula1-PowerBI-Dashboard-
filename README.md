# Formula 1 Analytics Dashboard (1950–2024)

## Overview
This project is a multi-page Power BI dashboard analyzing Formula 1 historical data from 1950 to 2024.

The dashboard provides:
- Driver performance analysis
- Constructor dominance analysis
- Circuit and race insights
- Historical trend analysis
- Geographic visualization of F1 circuits

---

## Dataset
Dataset Source:
Kaggle – Formula 1 World Championship (1950–2024)

Files used:
- results.csv
- drivers.csv
- races.csv
- constructors.csv
- circuits.csv

---

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

---

## Dashboard Pages

### 1. Overview Dashboard
- KPI Cards
- Top Drivers
- Top Constructors

### 2. Driver Analysis
- Driver trends
- Performance analysis

### 3. Circuit & Race Insights
- Global circuit map
- Race expansion trends
- Country-wise circuit analysis

---

## Key Insights
- Lewis Hamilton leads historically in total points
- Ferrari remains one of the most dominant constructors
- Formula 1 expanded significantly globally after 2000
- Modern F1 seasons contain significantly more races
